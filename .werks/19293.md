# messaging: RabbitMQ management web interface now only accessible via TLS

The RabbitMQ management web interface (enabled by the management plugin)
is now exclusively accessible via TLS.

Previously, the interface was exposed over plain HTTP/TCP.
It is now configured to use the site's existing TLS certificates.
